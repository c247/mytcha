Here’s what to send your frontend dev. It’s the **exact integration plan** (Auth + username claim + CRUD rankings + read others + read global).

---

## Firebase setup assumptions

* Auth enabled: **Email/Password**
* Firestore enabled
* Firestore rules already published (the ones we wrote)

---

## Data model (don’t change)

* `usernames/{usernameLower}` → `{ uid, createdAt }`
* `users/{uid}` → `{ username, createdAt }`
* `users/{uid}/rankings/{rid}` → `{ name, location, ranking, colour, updatedAt }`
* `global/leaderboards` → `{ top10: [...] }` (read-only from client)

## Frontend tasks

### 1) Add Firebase to React app

1. `npm i firebase`
2. Create `src/firebase.ts`:

```ts
import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";
import { getFirestore } from "firebase/firestore";

const firebaseConfig = {
  apiKey: import.meta.env.VITE_FIREBASE_API_KEY,
  authDomain: import.meta.env.VITE_FIREBASE_AUTH_DOMAIN,
  projectId: import.meta.env.VITE_FIREBASE_PROJECT_ID,
};

const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);
export const db = getFirestore(app);
```

3. Add env vars (Vite) in `.env`:

* `VITE_FIREBASE_API_KEY=...`
* `VITE_FIREBASE_AUTH_DOMAIN=...`
* `VITE_FIREBASE_PROJECT_ID=...`

(From Firebase Console → Project Settings → Web App config.)

### 3) Claim username (MUST be transactional)

When user signs up, after Auth returns `uid`, claim username:

```ts
import { doc, runTransaction, serverTimestamp } from "firebase/firestore";
import { db } from "./firebase";

export async function claimUsername(uid: string, usernameRaw: string) {
  const username = usernameRaw.trim().toLowerCase();
  if (!/^[a-z0-9_]{3,20}$/.test(username)) {
    throw new Error("Username must be 3-20 chars: a-z, 0-9, underscore.");
  }

  const unameRef = doc(db, "usernames", username);
  const userRef = doc(db, "users", uid);

  await runTransaction(db, async (tx) => {
    const existing = await tx.get(unameRef);
    if (existing.exists()) throw new Error("Username already taken.");

    tx.set(unameRef, { uid, createdAt: serverTimestamp() });
    tx.set(userRef, { username, createdAt: serverTimestamp() }, { merge: true });
  });

  return username;
}
```

UI flow:

* Signup form collects email/password + username
* If transaction fails (taken), show error and let user pick another

### 4) Write my rankings (create/update/delete)

Create:

```ts
import { collection, addDoc, serverTimestamp } from "firebase/firestore";
import { db } from "./firebase";

export async function addRanking(uid: string, r: {
  name: string; location: string; ranking: number; colour: string;
}) {
  await addDoc(collection(db, "users", uid, "rankings"), {
    ...r,
    updatedAt: serverTimestamp(),
  });
}
```

Update:

```ts
import { doc, updateDoc, serverTimestamp } from "firebase/firestore";

export async function updateRanking(uid: string, rid: string, patch: any) {
  await updateDoc(doc(db, "users", uid, "rankings", rid), {
    ...patch,
    updatedAt: serverTimestamp(),
  });
}
```

Delete:

```ts
import { doc, deleteDoc } from "firebase/firestore";

export async function deleteRanking(uid: string, rid: string) {
  await deleteDoc(doc(db, "users", uid, "rankings", rid));
}
```

### 5) Read my rankings (live)

```ts
import { collection, onSnapshot, orderBy, query } from "firebase/firestore";

export function subscribeMyRankings(uid: string, cb: (rows: any[]) => void) {
  const q = query(collection(db, "users", uid, "rankings"), orderBy("ranking", "asc"));
  return onSnapshot(q, (snap) => {
    cb(snap.docs.map(d => ({ id: d.id, ...d.data() })));
  });
}
```

### 6) Read someone else’s rankings by username (readonly)

```ts
import { doc, getDoc, collection, getDocs, orderBy, query } from "firebase/firestore";
import { db } from "./firebase";

export async function getRankingsByUsername(usernameRaw: string) {
  const username = usernameRaw.trim().toLowerCase();
  const unameSnap = await getDoc(doc(db, "usernames", username));
  if (!unameSnap.exists()) throw new Error("User not found.");

  const uid = unameSnap.data().uid as string;
  const q = query(collection(db, "users", uid, "rankings"), orderBy("ranking", "asc"));
  const rankingsSnap = await getDocs(q);
  return rankingsSnap.docs.map(d => ({ id: d.id, ...d.data() }));
}
```

Rules allow read for everyone, but writes only to your own `uid`.

---

### 7) Read global leaderboard

```ts
import { doc, getDoc } from "firebase/firestore";
import { db } from "./firebase";

export async function getGlobalTop10() {
  const snap = await getDoc(doc(db, "global", "leaderboards"));
  return snap.exists() ? snap.data().top10 : [];
}
```

---

## Notes / guardrails

* Username is **lowercase only** everywhere.
* Use `serverTimestamp()` not `new Date()`.
* Don’t attempt to write `/global/*` from the client (rules block it).
* Global top10 should be maintained by Cloud Function later.
